# Pop-Up Buoy  Project #

This repository contains the source code of the Pop-Up buoys from the PLOME project. 

The Pop-Up buoys connect to the Pop-Up Server and download its data. The Pop-Up Server source code can be found [here](https://github.com/obsea-upc/popup-server).

## Documentation ##

See [docs/ARCHITECTURE.md](docs/ARCHITECTURE.md) for firmware architecture, the buoy state machine, sequence diagrams for each mission phase, storage layout, and the full configuration reference. See [docs/TEST_PLAN.md](docs/TEST_PLAN.md) for the test suite plan.
